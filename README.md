# C# WebDriver Screenshots Extensions
[![nuget](http://flauschig.ch/nubadge.php?id=Noksa.WebDriver.ScreenshotsExtensions)](https://www.nuget.org/packages/Noksa.WebDriver.ScreenshotsExtensions/)
